## Troubleshooting

### Q: Getting "Connection refused" error?
**A:** 
Check:
1. Server is running
2. Correct host/port in .env
3. Firewall isn't blocking
4. Database is accessible

### Q: API key not working?
**A:** 
1. Verify key is active
2. Check expiration date
3. Verify correct permissions
4. Test with curl/Postman

### Q: Slow performance?
**A:** 
1. Check database indexes
2. Enable caching
3. Optimize queries
4. Review logs for errors
